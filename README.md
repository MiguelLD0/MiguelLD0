## Sobre mim

Sou estudante de Ciencia da Computacao, atualmente no 6º periodo, e gosto de aprender construindo projetos que parecem pequenos no comeco, mas logo viram sistemas completos com login, banco de dados, tempo real, arquivos, interface e varias decisoes tecnicas no meio do caminho.

Hoje meu foco principal esta em desenvolvimento web, backend, bancos de dados, comunicacao em tempo real e processamento de midia. Tambem venho estudando Python para analise de dados e automacao, alem de Java em projetos academicos.

Estou em fase de evolucao constante. Por isso, alem de fazer funcionar, tenho dado cada vez mais atencao para organizacao do codigo, modularizacao, seguranca, performance e experiencia do usuario.
